# AI Attendance System

An AI-powered attendance management system developed to automate attendance tracking using facial recognition and cloud-based data management.

---

## Live Demo

https://snapclass-ai-rohitha.streamlit.app/

---

## Overview

The system is designed to improve the efficiency and accuracy of attendance management by leveraging computer vision technologies. It enables automated student identification, real-time attendance processing, and secure data storage through cloud integration.

---

## Features

- Facial recognition-based attendance tracking
- Real-time face detection and processing
- Automated attendance recording
- Cloud database integration using Supabase
- Interactive user interface with Streamlit
- Efficient and scalable attendance management

---

## Technologies Used

- Python
- Streamlit
- OpenCV
- MediaPipe
- Supabase
