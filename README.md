# SnapClass AI Attendance System

An AI-powered attendance management system that automates attendance tracking using facial recognition technology and cloud-based data storage. The application enables real-time face detection, automated attendance recording, and efficient attendance management through a web-based interface.

## Live Demo

https://snapclass-ai-rohitha.streamlit.app/

---

## Overview

SnapClass AI is designed to streamline attendance management by leveraging Artificial Intelligence and Computer Vision. The system automatically identifies individuals, records attendance in real time, and securely stores attendance records in the cloud.

By eliminating manual attendance processes, the application improves accuracy, reduces administrative effort, and enhances operational efficiency.

## System Workflow

1. The user uploads or captures an image.
2. MediaPipe detects facial landmarks.
3. OpenCV processes facial features for identification.
4. The system matches the detected face with registered records.
5. Attendance is automatically marked.
6. Attendance data is stored in Supabase.
7. Results are displayed through the Streamlit interface.
